# Module 3B-5B Consensus Synthesis

## Status

This synthesis bridges the current B-phase evidence layers for Modules 3B, 4B, and 5B.

- Module 3B is a provisional seed-consensus layer from 7 immune/myeloid papers. Module 3A is active and not saturated.
- Module 4B is consensus-ready first pass from the saturated Module 4A corpus.
- Module 5B is a provisional seed-consensus layer from 9 axon/white-matter repair papers. Module 5A is active and not saturated.

## Cross-Module Consensus Drafts

| Consensus ID | Statement | Support | Boundary |
|---|---|---|---|
| M3-5-S001 | SCI repair mechanisms should be stratified into immune-state control, graft/cell replacement or modulation, and axon/white-matter repair rather than pooled as one regenerative therapy class. | M3B-S001-M3B-S006; M4B-S001-M4B-S008; M5B-S001-M5B-S006 | Modules 3B and 5B are seed-level; 4B is the only saturated B layer in this range. |
| M3-5-S002 | Myeloid cells can be harmful or beneficial depending on timing, state, and wound function; this directly constrains cell-transplant and axon-regeneration interpretations. | M3B-S001-M3B-S003; M5B-S003 | Broad anti-inflammatory conclusions are unsafe without timepoint and cell-identity extraction. |
| M3-5-S003 | Cell transplantation evidence is strongest when split by mechanism: neural relay/integration, glial or remyelinating support, trophic or immunomodulatory cells, and scaffold-cell combinations. | M4B-S001-M4B-S008 | Module 4B remains mostly abstract-level and needs full-text upgrades for quantitative effect sizes. |
| M3-5-S004 | Axon repair claims require separate labels for sprouting, true regeneration, remyelination, spared white matter repair, and functional recovery. | M5B-S001-M5B-S006; M4B-S002, M4B-S004 | Graft-enabled growth overlaps with Module 4 and should not be treated as cell-free regeneration. |
| M3-5-S005 | The strongest cross-module hypothesis is that durable SCI repair likely requires coordinated lesion permissiveness, immune-state regulation, and tract/circuit-specific growth support. | M3B boundary map; M4B boundary map; M5B boundary map | This is a synthesis hypothesis, not a proven consensus, until Module 3A and 5A reach saturation. |
