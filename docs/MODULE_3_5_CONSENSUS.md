# Module 3B-5B Consensus Synthesis

## Status

This synthesis bridges the current B-phase evidence layers for Modules 3B, 4B, and 5B.

- Module 3B is consensus-ready first pass from the saturated Module 3A immune/myeloid corpus.
- Module 4B is consensus-ready first pass from the saturated Module 4A cell-transplantation corpus.
- Module 5B is consensus-ready first pass from the saturated Module 5A axon/remyelination/white-matter corpus.

All three B layers remain first-pass and mostly abstract-level. Quantitative effect-size consensus should wait for full-text upgrades and figure-level extraction.
