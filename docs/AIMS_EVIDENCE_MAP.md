## Module Gaps

Module 13 has been added to cover the process of traumatic SCI and how it produces the chronic lesion architecture captured in Module 1. The aims logic is now covered by Modules 1-13, with Module 12 filling the broader innate immune conditioning, tolerance, and trained/tolerized host-state discovery layer. LPS is one mechanistic anchor inside that module rather than the module boundary.
