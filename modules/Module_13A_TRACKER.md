# Module 13A: Traumatic SCI Pathobiology and Chronic Lesion Formation - Corpus Building Tracker

## Overview

This tracker captures the process by which traumatic spinal cord injury evolves from the primary mechanical insult into the chronic lesion architecture tracked in Module 1. It is temporal and causal: the unit of interest is how hemorrhage, ischemia, blood-spinal cord barrier disruption, excitotoxicity, cell death, demyelination, inflammation, astrocytic border formation, fibrotic scar formation, cavitation, Wallerian degeneration, and tissue remodeling interact across acute, subacute, and chronic windows.

This module feeds Module 1 rather than duplicating it. Module 1 defines the chronic lesion endpoint; Module 13 defines the injury-to-lesion trajectory and mechanistic decision points that determine chronic anatomy.
